BeautyHouse 💄
BeautyHouse is a pet project built to demonstrate a basic backend system for managing beauty appointments. The application focuses on core functionality such as authentication and booking creation, following modern backend development practices.
✨ Features
* 🔐 Registration and authentication for beauty masters
* 🔑 JWT-based authentication
* 📅 Ability to create and manage appointments for beauty procedures
🏗 Architecture
The project is built with FastAPI and follows Clean Architecture principles, ensuring:
* Clear separation of concerns
* Scalable and maintainable codebase
* Easy testing and extensibility
🧪 Testing
The project includes a comprehensive testing approach:
* ✅ Unit tests
* 🔗 Integration tests
* 🚀 End-to-end (E2E) tests
🛠 Tech Stack
* Backend: FastAPI
* Authentication: JWT
* Database: PostgreSQL
* Testing: pytest

  📌 Purpose
This project was created as a learning and portfolio project to practice:
* Backend development with FastAPI
* Designing systems using Clean Architecture
* Implementing authentication mechanisms (JWT)
* Writing different levels of automated tests
🔮 Future Improvements
* Role-based access control
* Advanced booking management
* Notifications system
